Engagement - Structural Tags

  1. What is the root? What is the relationship between folder and file names and the path?
    The root is the top (starting) folder of your website/project. All paths start from there.
  2. What is a default document?
    A default document is the file a web server automatically loads when you visit a folder URL. Most commonly it’s index.html.
  3. Explain the difference between an absolute link and a relative link. Why would you use relative links?
    Absolute link: the full web address (starts with http:// or https://). Relative link: a link based on your current page’s location (no domain name).

  5. List the 3 rules with relative links.
    Same folder: use just the filename
    Down a folder: use foldername/filename
    Up a folder: use ../ to go up one folder (repeat if needed)

  7. Do links always need to be inside a NAV tag?
