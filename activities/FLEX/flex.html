Engagement CSS Flex

  1. What you learned about Flex and CSS.
      I learned that applying display: flex; to a parent container allows it to control the layout of its child elements. In this project, I used justify-content: space-evenly; to space navigation links and flex-flow: row wrap; to organize cards into flexible rows that adjust automatically.
  2. How far did you get on your Flex Box Froggy Game or What did you learn on your own?
      I practiced using properties like justify-content, flex-direction, and order to control alignment and positioning. Through practice, I better understood how flex containers manage spacing, wrapping, and the visual order of elements without changing the HTML structure
