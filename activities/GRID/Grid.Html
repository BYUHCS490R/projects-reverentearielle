Engagement

  1. Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid:
      The parent is the container element and the children are the items inside it. Flex and grid use the parent to control how the children lay out. So you must put items inside the right parent to make the layout behave the way you want.
  2. What are the two main differences between CSS Grid and CSS Flex?
      Grid is for two dimensional layouts rows and columns and Grid lets you place items by row and column positions. Flex is for one dimensional layouts a row or a column at a time and Flex flows items and controls spacing along a single axis.
