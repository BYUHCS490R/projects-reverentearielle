  2. What is normal flow in HTML?
      Normal flow is the default way elements are placed on a page block elements stack on top of each other and inline elements sit next to each other
  3. What does float do? Why do you need to clear the float?
      Float moves an element to the left or right so text can wrap around it you need to clear the float so other elements do not wrap around it and layout stays correct
  4. What do absolute, fixed, and relative do?
      Relative moves an element from its normal position absolute places an element based on its nearest positioned parent fixed keeps an element in the same spot on the screen even when you scroll
  5. How can you make nice navigation buttons for mobile devices?
      You can use CSS styling with padding background color border radius and media queries to make buttons bigger easy to tap and responsive
