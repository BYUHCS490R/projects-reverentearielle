Engagement

  1. What is the difference between block tags and inline tags?
      Block tags start on a new line and take the full width available like <div>, <p>, <h1> Inline tags stay in the flow and only take the space they need like <span>, <a>, <img>
  2. What is the margin, border, and padding?
      Margin is the outer space between elements border is the line around an element padding is the inner space between the content and the border
  3. What are units of measurement you can use in CSS?
      Common units are px em rem % vw vh and physical units like cm mm in and print units like pt
  4. When would you use a figure tag?
      Use <figure> to group an image or other media with a caption using <figcaption> for things like photos diagrams or code output that need a label
