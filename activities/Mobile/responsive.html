Engagement

  1. Why do we need a viewport meta tag?
      It tells the phone browser to use the device width for the page so fonts and layout scale correctly. without it the page looks tiny or zoomed out on mobile.
  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?
      The rule that appears later in the CSS file wins, so the second one will apply unless the other rule is more specific or uses !important.
  3. How do you change the CSS for a web page displayed on a phone
      Use a media query like @media (max-width: 600px) and inside it change layout, font sizes, hide or stack elements so the page fits and is easy to tap.
