  1. Why is the viewport meta tag important?

  2. How can you adjust the CSS for printing?When would you want to adjust the CSS for printing?
      Use an @media print stylesheet to hide navigation, remove heavy backgrounds, increase font size for readability, and add page-break rules; you want this when people will print the page so the printed version is clean and readable.
  3. When would you want to adjust the CSS for printing?
      Adjust CSS for printing any time you expect users to make a paper copy for handouts, receipts, reports, or homework so printed output looks correct and excludes web-only elements.
