Engagement

  1. List out the different ways to select an item.
    • Element/tag selector: p h1 img
    • Class selector: .box
    • ID selector: #title
    • Group selector: h1, h2, p
  2. What is the difference between an ID and a Class?
    • ID is for one unique element on the page (#oneThing),
    • Class is for many elements that share the same style (.manyThings).
  3. What is the difference between a SPAN tag and a DIV tag?
    • SPAN is inline that stays in the same line, for small pieces of text.
    • DIV is block that starts on a new line for sections/layout
  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?
      DIV is a general container with no meaning while header, main, footer are semantic containers that tell what the section is for.
  5. What are the different ways to select a color?
    • Color names: red, blue
    • Name
    • RGB: rgb(255, 0, 0)
    • Hex code
    • Short hexcode
