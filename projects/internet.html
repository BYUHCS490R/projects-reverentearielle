Internet

The Internet is a global network of computers that allows people and devices to communicate and share information

Web

The Web is a part of the Internet that lets people view websites using a web browser.

How are the Internet and Web different?

The Internet is the network itself, while the Web is a service on that network used to access web pages.

ASCII and Unicode

ASCII is an older system that represents basic English characters, while Unicode supports characters from many languages.

What is UTF-8

UTF-8 is a common way to store Unicode characters so computers can read and share text correctl

URL

A URL is the address used to find a page or resource on the Internet.

Domain Name

A domain name is the easy-to-read name of a website, like google.com.

HTTP

HTTP is the set of rules that allows browsers and servers to send web pages.

HTTPS

HTTPS is a secure version of HTTP that protects data while it is being sent.

IP Address

An IP address is a number that identifies a device on a network/p>

DNS

DNS translates domain names into IP addresses so websites can be found.

W3C Recommendations

W3C Recommendations are rules that help make websites work correctly across browsers.

XML, XHTML, HTML, HTML5

XML stores data, XHTML is strict HTML, HTML builds web pages, and HTML5 is the newest version with modern features.
